Paddle dashboard chat widget browser requirements

The Paddle dashboard chat widget support the following browsers:

  * Google Chrome: the two most recent stable releases
  * Apple Safari: the two most recent stable releases
  * Microsoft Edge: the two most recent stable releases
  * Mozilla Firefox: the two most recent stable releases
  * Microsoft Internet Explorer (IE): Version 11


Additionally, do note that some privacy browser extensions/plugins such as Ghostery may interfere with the chat widget. If you're unable to see the chat widget button, please try disabling or whitelisting the Paddle dashboard domain and refresh the page.
